# 人事管理系统简介 java347

## 项目概述

人事管理系统是一款集员工信息管理、考勤管理、薪酬管理等功能于一体的企业内部管理工具。该系统主要服务于企业的人力资源管理部门，通过简化管理流程，提高管理效率。

## 技术栈

- 前端：Vue2 + ElementUI
- 后端：Spring Boot
- 数据库：MySQL 5.7及以上
- 开发工具：IDEA、VSCode
- Node.js版本：14

## 功能模块

### 系统角色

- 管理员
- 员工

#### 管理员角色功能

- 个人中心：查看和管理个人相关信息
- 部门信息管理：添加、修改、删除部门信息
- 员工信息管理：添加、修改、删除员工信息
- 考勤信息管理：管理员工考勤信息
- 上班记录管理：管理员工上班记录
- 下班记录管理：管理员工下班记录
- 加班记录管理：管理员工加班记录
- 员工薪酬管理：管理员工薪酬

#### 员工角色功能

- 个人中心：查看和管理个人相关信息
- 考勤信息管理：查看和管理个人考勤信息
- 上班记录管理：查看个人上班记录
- 下班记录管理：查看个人下班记录
- 加班记录管理：查看个人加班记录
- 员工薪酬管理：查看和管理个人薪酬信息

## 运行环境

## 核心亮点

- 前后端不分离，统一技术栈，方便维护和迭代
- 使用Spring Boot快速构建后端服务，提高开发效率
- 采用Vue2 + ElementUI搭建的前端界面，响应式设计，操作便捷
- 完善的权限管理，确保系统安全可靠

## 部署步骤

1. 准备运行环境：JDK 1.8、MySQL 5.7及以上
2. 克隆代码仓库到本地
3. 编译前端代码：`cd frontend && npm install && npm run build`
4. 编译后端代码：`cd backend && mvn clean package`
5. 部署应用到服务器，启动后端服务
6. 访问前端页面，开始使用系统

注意：以上步骤仅供参考，实际部署时请根据具体环境进行调整。
